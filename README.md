# WAV播放器 Pro Max

這是一個使用 C# 與 Windows Forms (WinForms) 開發的進階 WAV 音效播放器，利用 Windows 內建的 MCI (Media Control Interface) 播放音訊，並搭載了現代化使用者介面與多項強大功能，無需安裝額外的 NuGet 套件。

## 全新現代化 UI
- **深色主題 (Dark Theme)**：全新設計的舒適深色配置，降低視覺疲勞。
- **漸層與圓角設計**：導入現代化的漸層按鈕、圓角卡片面板，提升整體質感。
- **波形圖預覽 (Waveform View)**：即時繪製 WAV 檔案的音訊波形，支援點擊波形進行精準進度跳轉 (Seek)。
- **動態音訊視覺化 (Live Visualizer)**：播放時顯示即時跳動的虛擬音訊條，增添播放樂趣。
- **統計儀表板 (Dashboard)**：提供音樂庫的檔案數量、總長度、總大小、格式分布圖 (Mini Bar Chart) 及其他詳細統計資訊。

## 核心功能特色 (Features)

1. **進階播放清單管理**
   - 支援加入單一/多個 WAV 檔案及整個資料夾。
   - 支援直接拖曳 WAV 檔案或資料夾至播放清單。
   - **我的收藏 (Favorites)**：可針對單曲標記星星收藏，並支援「只顯示收藏」的篩選。
   - **即時搜尋篩選 (Real-time Search)**：輸入關鍵字立即過濾播放清單中的檔案。
   - **資料夾監看 (Folder Watcher)**：可指定監看特定資料夾，有新 WAV 檔案加入時會自動載入播放清單。
   - 支援清除失效/遺失的檔案連結。
   - 儲存 (Save) 與載入 (Load) M3U / TXT 格式的播放清單。

2. **進階播放控制**
   - 包含播放、暫停、停止、上一首、下一首、隨機播放 (Shuffle)。
   - 支援循環模式：不循環、單曲循環、清單循環。
   - **A/B 點循環播放 (A/B Looping)**：可自由設定起點 (A) 與終點 (B)，自動在該區段內反覆播放。
   - 支援靜音切換 (Mute)。

3. **音訊調整**
   - 支援音量調整 (0% ~ 100%)。
   - 支援播放速度/倍速調整 (50% ~ 200%)。

4. **強大的音訊工具箱 (Audio Toolbox)**
   - **檢查完整性**：掃描清單所有檔案的格式與長度狀態大。
   - **匯出報表**：支援匯出播放清單與音訊資訊為 CSV（試算表）或 HTML（圖表網頁）格式。
   - **靜音段偵測**：自動掃描目前音訊，找出超過 0.8 秒的明顯靜音片段。
   - **簡易 WAV 編輯與特效 (不依賴外部套件)**：
     - **反轉 (Reverse)**：產生倒轉播放的 WAV。
     - **淡入淡出 (Fade In/Out)**：套用 1 秒的起頭與結尾音量漸變。
     - **音量正規化 (Normalize)**：自動掃描並提升整體音量而不破音。
   - **快速備份**：產生目前檔案的備份副本。

5. **WAV 資訊深度解析**
   - 讀取 WAV 檔標頭 (RIFF Header) 解析出音訊格式 (支援 PCM / 32-bit IEEE Float)。
   - 精確顯示聲道數 (Channels)、取樣率 (Sample Rate)、位元深度 (Bits Per Sample)、長度與檔案大小。

6. **自動記憶狀態 (Session Save/Load)**
   - 關閉程式時自動記憶當前的播放清單、音量、倍速、循環模式與收藏紀錄，下次開啟自動還原。

7. **快捷鍵支援**
   - `Space` (空白鍵)：暫停 / 繼續播放
   - `Ctrl + O`：開啟加入檔案對話框
   - `Ctrl + F`：跳轉至搜尋框
   - `Right` (右方向鍵)：快轉 5 秒
   - `Left` (左方向鍵)：倒退 5 秒
   - `N`：切換至下一首
   - `P`：切換至上一首
   - `Enter`：播放目前選取歌曲
   - `Delete`：移除播放清單中選取的項目
