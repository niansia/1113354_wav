# 音訊播放器

這是一個使用 C# 與 Windows Forms (WinForms) 開發的進階桌面音訊播放器，支援 **WAV / MP3 / MP4** 檔案播放，並提供波形預覽、播放清單管理、我的最愛、近期播放、排行榜、A/B 循環、音訊工具箱與統計儀表板等功能。

本專案以 Windows 內建播放能力為核心，WAV 使用 MCI (Media Control Interface) 播放，MP3 / MP4 則使用 Windows Media Player COM 元件播放，並保留不需額外安裝 NuGet 套件的設計。

---

## 專案特色

- **支援多格式播放**：支援 WAV、MP3、MP4 檔案。
- **不需額外 NuGet 套件**：主要依賴 Windows 內建元件與 WinForms。
- **Spotify / Apple Music 風格 UI**：採用深色介面、側邊導覽、內容卡片與底部播放列。
- **波形圖預覽**：WAV 可讀取實際 PCM 波形，MP3 可產生預覽波形。
- **完整播放清單功能**：支援加入檔案、加入資料夾、拖曳加入、搜尋、收藏與 M3U / TXT 匯入匯出。
- **播放紀錄與排行**：自動記錄播放次數與最近播放時間，可切換近期播放與排行榜。
- **A/B 循環播放**：可設定 A 點與 B 點，在指定區段內反覆播放。
- **音訊工具箱**：提供完整性檢查、報表匯出、靜音段偵測與簡易 WAV 編輯功能。
- **自動儲存狀態**：關閉程式後可記住播放清單、音量、倍速、循環模式、收藏與播放紀錄。

## 全新現代化 UI

本播放器已從傳統工具型介面調整為更接近音樂平台的操作體驗：

- **深色主題 (Dark Theme)**：使用黑色與深灰色作為主背景，降低視覺疲勞。
- **Spotify 風格側邊欄**：左側提供播放清單、我的最愛、近期播放、排行榜等導覽功能。
- **首頁快速卡片**：提供「我的最愛」、「近期播放」、「排行榜」、「加入音訊」等快速入口。
- **底部播放控制列**：將播放、暫停、上一首、下一首、音量、倍速與波形圖集中在底部。
- **右側正在播放資訊**：顯示目前播放狀態、檔案資訊與檔案位置。
- **圓角與漸層卡片**：改善整體視覺層次與現代感。
- **動態音訊視覺化 (Live Visualizer)**：播放時顯示跳動的視覺化效果。
- **統計儀表板 (Dashboard)**：顯示音樂庫檔案數量、總長度、總大小與格式分布等資訊。

## 核心功能特色

### 1. 多格式播放支援

- 支援加入與播放：
  - `.wav`
  - `.mp3`
  - `.mp4`
- 支援單一檔案、多檔案與整個資料夾匯入。
- 支援拖曳檔案或資料夾到播放器中。
- 支援 MP4 音軌播放，但不顯示影片畫面。

---

### 2. 播放清單管理

- 支援播放清單檢視。
- 支援加入音訊 / 影片檔案。
- 支援加入整個資料夾並自動掃描子資料夾。
- 支援即時搜尋篩選。
- 支援清除失效或遺失的檔案。
- 支援 M3U / TXT 播放清單匯入。
- 支援儲存 M3U 播放清單。
- 支援右鍵選單：
  - 播放
  - 切換收藏
  - 在檔案總管中顯示
  - 複製路徑
  - 移除

---

### 3. 我的最愛

- 可將歌曲加入我的最愛。
- 可快速切換至「我的最愛」頁面。
- 可使用「只顯示收藏」篩選目前清單。
- 收藏紀錄會自動儲存，下次開啟程式時自動還原。

---

### 4. 近期播放

- 播放歌曲時會自動記錄最近播放時間。
- 可切換至「近期播放」查看最近聽過的歌曲。
- 近期播放紀錄會自動儲存。
- 可依最近播放順序快速找回剛聽過的歌曲。

---

### 5. 排行版

- 每次播放歌曲時會自動累計播放次數。
- 可切換至「排行榜」查看播放次數最高的歌曲。
- 排行榜會依播放次數排序，播放次數相同時依最近播放時間排序。
- 播放次數與最近播放時間會顯示在播放清單中。

---

### 6. 播放控制

- 播放
- 暫停 / 繼續
- 停止
- 上一首
- 下一首
- 隨機播放 (Shuffle)
- 循環播放：
  - 不循環
  - 單曲循環
  - 清單循環
- 靜音切換
- 點擊波形或拖曳進度條可跳轉播放位置。

---

### 7. A/B 循環播放

可自由設定指定區段進行循環播放：

- 設定 A 點
- 設定 B 點
- 清除 A/B 區間
- 播放到 B 點後會自動跳回 A 點

適合用於：

- 語音學習
- 聽寫練習
- 音樂片段重複聆聽
- 音訊檢查

---

### 8. 音訊調整

- 支援音量調整。
- 支援靜音。
- 支援播放速度 / 倍速調整：
  - 50% ~ 200%

> 備註：不同格式與 Windows 播放元件對倍速支援程度可能不同。

---

### 9. 波形圖與視覺化

- WAV 檔案可讀取 PCM 資料產生較精準的波形。
- MP3 檔案可產生預覽用波形。
- 波形圖支援點擊跳轉播放位置。
- 播放時會顯示目前播放位置。
- Live Visualizer 會依照播放狀態呈現動態視覺效果。

---

### 10. 音訊工具箱

提供多項輔助工具：

- **檢查完整性**：掃描播放清單中的檔案狀態、格式與長度。
- **匯出 CSV 報表**：將播放清單資訊匯出為 CSV。
- **匯出 HTML 報表**：將播放清單與統計資訊匯出為 HTML。
- **偵測目前檔案靜音段**：找出明顯靜音片段。
- **反轉目前 WAV**：產生反向播放的 WAV 檔案。
- **淡入淡出目前 WAV**：套用開頭與結尾淡入淡出效果。
- **音量正規化目前 WAV**：提升整體音量並避免破音。
- **備份目前檔案**：快速建立目前檔案的備份副本。

> 備註：WAV 編輯功能主要針對 WAV PCM 檔案設計。

---

### 11. 音訊資訊解析

可顯示檔案資訊：

- 檔名
- 長度
- 格式
- 取樣率
- 聲道數
- 位元深度
- 檔案大小
- 播放次數
- 最近播放時間
- 檔案路徑

WAV 檔案可解析 RIFF Header，取得 PCM / IEEE Float 等資訊。

---

### 12. 自動記憶狀態

關閉程式時會自動記憶：

- 播放清單
- 我的最愛
- 音量
- 倍速
- 隨機播放狀態
- 循環模式
- 監看資料夾
- 播放次數
- 最近播放時間
- 近期播放紀錄

下次開啟程式會自動還原。

## 快捷鍵支援

| 快捷鍵 | 功能 |
|---|---|
| `Space` | 暫停 / 繼續播放 |
| `Ctrl + O` | 開啟加入檔案對話框 |
| `Ctrl + F` | 跳轉至搜尋框 |
| `Right` | 快轉 5 秒 |
| `Left` | 倒退 5 秒 |
| `N` | 下一首 |
| `P` | 上一首 |
| `Enter` | 播放目前選取歌曲 |
| `Delete` | 移除播放清單中選取項目 |

## 注意事項

- 本專案不需要額外安裝 NuGet 套件。
- MP3 / MP4 播放依賴 Windows Media Player COM 元件。
- 若 Windows 未啟用 Media Features，MP3 / MP4 播放可能會失敗。
- MP4 支援音軌播放，不支援影片畫面顯示。
- WAV 編輯工具主要支援 WAV PCM 格式。
- 部分特殊編碼的 MP3 / MP4 可能因 Windows 系統解碼器支援度而無法正常播放或產生波形。
